Kelly Criterion (标准版)

资金管理的圣杯。它不告诉你该买什么，只告诉你该买多少。它能算出在当前胜率和赔率下，为了最大化复利增长且不破产，你该下注多少比例的仓位。

// 标准公式 (针对单笔交易)
K% = W - [ (1 - W) / R ]
SOP 3.0 实战解读

• W = 胜率 (Win Rate)

• R = 盈亏比 (Avg Win / Avg Loss)

• 注意： 此结果为“单笔建议仓位”。

• 如显示 20% 且你开 5 个币，总仓位即 100%。请务必自行把控总风险。

Current Streak (连胜/连败)

心理状态温度计。显示你当前是处于“顺风局”（手热）还是“逆风局”（上头）。它是防止报复性交易的第一道防线。

IF Win: Streak + 1 (正数)
IF Loss: Streak = -1 (负数)
SOP 3.0 熔断机制

• +3 连胜: 警惕膨胀。建议减仓或提现，防止把利润还给市场。

• -2 连败: 立即停手。关软件休息 4 小时，这是最容易发生报复性交易的时刻。

SQN = (Expectancy / StdDev) * √N

Win Rate & R/R Ratio

交易的左膀右臂。这两个指标必须结合看。

Win Rate (胜率)
Win / Total_Trades

SOP 目标: > 40%

R/R Ratio (盈亏比)
Avg_Win / Avg_Loss

SOP 目标: > 2.0

Profit Factor (盈利因子)

每亏 1 美元，能赚回多少？

PF = Gross_Win / Gross_Loss

• < 1.0: 亏钱系统。

• 1.0 - 1.5: 勉强盈利。

• > 2.0: 健康 (SOP 3.0 目标)。

Expectancy (单笔期望)

每次开单的平均预期收益。

EV = (Win% × Avg_Win) - (Loss% × Avg_Loss)

只要 EV > 0，长期坚持执行即可获利。

Max Drawdown (最大回撤)

疼痛指数。资产从最高点跌到最低点的幅度。

MDD% = (Peak - Valley) / Peak

• 亏 50%，需涨 100% 回本。

• SOP 3.0 红线: 20%。
