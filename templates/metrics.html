TRADES 交易指标 基于历史交易记录计算

Kelly Criterion 凯利公式

资金管理的圣杯。它不告诉你该买什么，只告诉你该买多少。它能算出在当前胜率和赔率下，为了最大化复利增长且不破产，你该下注多少比例的仓位。

// 标准公式 (针对单笔交易)
K% = W - [ (1 - W) / R ]
SOP 3.0 实战解读

• W = 胜率 (Win Rate)

• R = 盈亏比 (Avg Win / Avg Loss)

• 注意： 此结果为"单笔建议仓位"。

• 如显示 20% 且你开 5 个币，总仓位即 100%。请务必自行把控总风险。

Current Streak 连胜/连败

心理状态温度计。显示你当前是处于"顺风局"（手热）还是"逆风局"（上头）。它是防止报复性交易的第一道防线。

IF Win: Streak + 1 (正数)
IF Loss: Streak = -1 (负数)
SOP 3.0 熔断机制

• +3 连胜: 警惕膨胀。建议减仓或提现，防止把利润还给市场。

• -2 连败: 立即停手。关软件休息 4 小时，这是最容易发生报复性交易的时刻。

Profit Factor 盈利因子

每亏 1 美元，能赚回多少？

PF = Gross_Win / Gross_Loss

• < 1.0: 亏钱系统。

• 1.0 - 1.5: 勉强盈利。

• > 2.0: 健康 (SOP 3.0 目标)。

Max Single Loss 单笔最大亏损

风控底线。历史上单笔交易造成的最大资金损失金额。

Max_Loss = Min(PnL_Net)

• 关注单笔亏损是否超过 预期风险。

• 避免出现 "一单回到解放前" 的情况。

System Quality (SQN) 系统质量

衡量赚钱的稳定性。如果你赚了很多钱但波动巨大（如赌博），SQN 会很低。

SQN = (Expectancy / StdDev) * √N
< 1.6
很难做
2.0 - 3.0
优秀
> 3.0
圣杯级

Win Rate & R/R Ratio 胜率与盈亏比

交易的左膀右臂。这两个指标必须结合看。

Win Rate (胜率)
Win / Total_Trades

SOP 目标: > 40%

R/R Ratio (盈亏比)
Avg_Win / Avg_Loss

SOP 目标: > 2.0

Expectancy (EV) 单笔期望

每次开单的平均预期收益。

EV = (Win% × Avg_Win) - (Loss% × Avg_Loss)

只要 EV > 0，长期坚持执行即可获利。

ROI 投资回报率

衡量投资效率。相对于初始本金，你赚了多少百分比。

ROI = (Total_PnL / Initial_Capital) × 100%

ROI 越高，说明资金利用效率越高。但也要结合风险指标一起看。

SHORT SETUPS 技术面做空经典图形 形态识别 · 触发 · 失效

Setup Cheat Sheet 做空速查表
短线规则
触发

• 破位后确认（收盘/放量/反抽失败）。

• 优先“第二次机会”而非第一次冲动。

止损

• 放在形态失效点（高点/上沿）。

• 止损必须小于结构性目标空间。

失效

• 价格重新站回关键结构并走高。

• 结构被“吞没”或量能反向扩张。

Double Top 双顶

反转 高位形态

高位反转信号。价格两次冲高失败，形成明显颈线支撑。

交易要点

• 触发：颈线下破收盘。

• 失效：收回颈线上方。

• 止损：右顶高点上方。

• 适用：高位或涨势末端更可靠。

Head & Shoulders 头肩顶

反转 高位形态

趋势反转经典形态。右肩弱于头部，颈线承压。

交易要点

• 触发：颈线下破反抽失败。

• 失效：站回颈线并走高。

• 止损：右肩高点上方。

• 适用：右肩显著缩量更稳。

Rising Wedge 上升楔形

反转 趋势末端

涨不动的上涨。价格抬高但斜率减弱，成交量常缩减。

交易要点

• 触发：跌破下沿趋势线。

• 失效：突破上沿并站稳。

• 止损：楔形上沿或近高。

• 适用：上升斜率明显放缓。

Bear Flag 熊旗

中继 趋势中

下跌中继。先急跌形成旗杆，随后窄幅反弹或横盘。

交易要点

• 触发：跌破旗形下沿。

• 失效：突破旗形上沿。

• 止损：旗形上沿上方。

• 适用：旗形回撤不过深。

Bull Trap 多头陷阱

反转 诱多

假突破诱多。冲破阻力后快速跌回，形成反向信号。

交易要点

• 触发：跌回突破位下方。

• 失效：重新站上突破位。

• 止损：假突破高点上方。

• 适用：假突破快速回撤。

Range Breakdown 区间下破

延续 区间破位

支撑失效。长时间横盘后向下破位，趋势开始加速。

交易要点

• 触发：支撑下破收盘。

• 失效：收回区间并站稳。

• 止损：原支撑或中轴上方。

• 适用：横盘时间越长越强。

MONITOR 监控指标 基于实时资金曲线计算

Drawdown 回撤

风险的核心度量。从峰值下跌了多少百分比。回撤越大，恢复越难。

// Current Drawdown 当前回撤
DD = (Peak_Equity - Current_Equity) / Peak_Equity × 100%
// Max DD (Period) 周期最大回撤
Max_DD = Max(DD) in selected period
回撤恢复难度

• 10% 回撤: 需要 11.1% 收益恢复

• 20% 回撤: 需要 25% 收益恢复

• 50% 回撤: 需要 100% 收益恢复

Health Score 账户健康度

综合评估账户状态的指标。结合当前回撤和波动率，评估账户的整体健康程度。

// 计算公式
Score = 100 - (Drawdown × 5) - (Volatility × 2)
// 结果限制在 0 ~ 100 之间
健康度等级

• > 80: 健康 (绿色) - 账户状态良好

• 50 - 80: 警戒 (黄色) - 需要关注风险

• < 50: 危险 (红色) - 建议减仓或停止交易

Sharpe Ratio 夏普比率

风险调整后的收益。每承担一单位风险，能获得多少超额收益。

Sharpe = (Mean_Return / StdDev) × √(Periods_Per_Year)
< 1.0
较差
1.0 - 2.0
良好
> 2.0
优秀

Calmar Ratio 卡玛比率

收益与最大回撤的比值。衡量你为获得收益承受了多大的痛苦。

Calmar = Annualized_Return / Max_Drawdown
< 1.0
较差
1.0 - 2.0
良好
> 2.0
优秀

Volatility 波动率

资金曲线的颠簸程度。波动率越低，曲线越平滑，心态越稳定。

Volatility = StdDev(Period_Returns) × 100%

• 越低越好：低波动意味着稳定盈利

• 高波动可能带来高收益，但也意味着高风险

Alpha Velocity 收益速度

每小时的预估收益。基于近期数据推算的盈利速度。

// 基于最近 60 个数据点 (约1小时)
Velocity = (Current - Start) / Time_Span × 60

正值表示资金在增长，负值表示资金在流失。仅供参考，不代表未来表现。

Win Rate (Time) 时间胜率

净值增长的时间占比。不同于交易胜率，这衡量的是资金曲线向上的时间比例。

Time_Win_Rate = Positive_Returns_Count / Total_Periods × 100%

高时间胜率意味着资金曲线更平滑向上，心理压力更小。

Peak Equity 峰值净值

历史最高点。累计净值曾经达到的最高值，是计算回撤的基准。

Peak = Max(Cumulative_Equity) in selected period

每次创新高都值得庆祝，但更重要的是守住利润，控制回撤。

Daily Trading SOP 实盘执行时间轴

晨间开单 + 午间风控 + 夜间压仓 + 次日资金再分配 的循环流程。

05:00-08:00 (UTC+8) // Entry Window
标的池：当日涨幅榜 Top10 + 14D 反弹榜 Top10 + 昨日Top10今日仍上涨延续池。
每标的固定 3000U，2x 杠杆，逐仓做空。
{{ noon_loss_check_time_label }} // Noon Risk Cut
检查全部持仓，浮亏仓位全部平掉；其余持有。
23:00 // Overnight Check
除非整体行情显著偏弱，过夜持仓控制在不超过 5 个标的。
Next 05:00-08:00 // Legacy Position Rebalance
旧仓位调至最大杠杆，先设定保护性止盈/出场锚点，再尽量降低保证金占用，释放资金给新仓。
Execution Notes

• 核心优势：节奏固定、执行清晰、资金利用率高。

• 主要风险：标的池同质化偏高，单边行情下做空逆风概率上升。

• 关键前提：升杠杆与减保证金动作，必须在保护单有效后执行。

做空加仓策略 逆势补仓推演表

加仓推演表 Averaging Down Strategy

逆势加仓的数学推演。当做空后价格逆向上涨时，分阶段补仓降低持仓均价，等待回调止盈离场。

// 前置设定
初始价格: 1.00
每次加仓: 3000U 仓位 (1500U 本金)
爆仓红线: 均价 +43%
加仓策略: 爆仓价前 3%-5%
阶段 累计本金 累计仓位 ⚖️ 持仓均价 💀 爆仓价 🛠️ 加仓点 🎯 止盈价 状态描述
1 1,500 U 3,000 U 1.000 1.430 -- 0.800 初始：跌20%赚600U
2 3,000 U 6,000 U 1.166 1.667 1.400 (+40%) 0.933 第1次救火：需回调到0.933赚1200U
3 4,500 U 9,000 U 1.282 1.833 1.600 (+60%) 1.197 第2次救火：回调到1.197赚600U离场
4 6,000 U 12,000 U 1.381 1.974 1.800 (+80%) 1.312 最终防线：回调到1.31安全撤退
⚠️ 风险提示

• 止盈价是按照第3、4阶段降低预期计算，只求赚 600U 总利润安全离场。

• 加仓点设置在爆仓价前 3%-5%，留有安全边际。

• 此策略适用于趋势判断正确但入场时机不佳的情况，不适用于趋势反转。

• 最多4次加仓，超过则需止损认错，不可无限补仓。
